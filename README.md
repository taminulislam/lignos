# LIGNOS

**Lignin-IL Gated Neural Orchestrated Specialists — a calibrated multimodal ensemble for ionic-liquid lignin-dissolution prediction**

LIGNOS predicts lignin-dissolution yield (and seven auxiliary thermodynamic
properties) for ionic-liquid (IL) solvents. It fuses three architecturally
distinct specialists — a SMILES graph baseline (D-MPNN), a DFT-surface +
vision-transformer specialist, and a COSMO-SAC σ-profile specialist — through a
24-parameter scalar **Bayesian model-averaging (BMA)** router, and adds a
**Stage-2 lignin head** conditioned on the model's own predicted thermodynamic
outputs (*cross-task feature distillation*).

Its central result is mechanistic: the same distillation head that raises
in-distribution lignin R² by ≈+0.05 lowers strict leave-IL-out R² by ≈−0.59,
isolating where within-network stacked generalization helps versus harms, and
motivating a confidence-gated conditional-deployment policy.

This repository contains the **model and pipeline source code** accompanying the
manuscript (under review). It does **not** include the experimental dataset,
trained checkpoints, or raw logs — see [Data & model availability](#data--model-availability).

> **Note on module naming.** A few model files under `lignos/models/`
> (`cosmobridge_v5.py`, `cosmobridge_v5_1.py`, `cosmobridge_v5_lite.py`) and their
> `COSMOBridgeV5` classes retain the name of the multimodal fusion architecture
> that LIGNOS is built on; the names are kept so the published class references
> match the code.

```

    train_a5_bma_stage2.py       Stage-2 cross-task distillation head
    train_a2_two_stage*.py       shared A2 backbone
    build_cosmo_sac_features.py  COSMO-SAC σ-profile descriptors

```

## Installation

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

Key dependencies: PyTorch ≥ 2.0, RDKit, Chemprop, Transformers (ViT), XGBoost,
scikit-learn, NumPy/Pandas. The DFT COSMO surface / σ-profile pipeline
additionally requires Psi4 and/or NWChem and xtb (GFN2-xTB); install those
separately per your platform.

## Data & model availability

To respect collaborator data-sharing terms, this repository ships **code only**:

- **Experimental dataset** (the LignoIL benchmark — activity coefficients,
  excess properties, and lignin-dissolution yields) — not included. Available
  from the authors on reasonable request.
- **Trained checkpoints** (`checkpoints/`) — not included (regenerate via the
  training scripts).
- **DFT surfaces / σ-profile banks / point clouds, logs, results, figures** —
  not included (regenerated by the pipeline and figure scripts).

Paths under `lignos/data/`, `lignos/checkpoints/`, and
`results/` are expected at runtime; point the configs at your own copies.

## Reproducing the main result

1. Assemble features: `build_cosmo_sac_features.py`, `build_il_vit_bank.py`,
   `extract_point_clouds_ligno.py` (DFT surfaces required).
2. Train the shared A2 backbone and specialists, then the BMA router:
   `train_a5_bma_pipeline.py`.
3. Train the Stage-2 cross-task distillation head: `train_a5_bma_stage2.py`.
4. Evaluate Task 1 (same-chemistry) and Task 2 (leave-IL-out): `eval_*.py`,
   `compare_a59_*` and the bootstrap scripts.
5. Virtual screen + σ-profile re-rank: `scripts/screen/screen_novel_ils.py`,
   `scripts/screen/rerank_gasteiger_sigma.py`.
